## 実装ノート

### デザインシステム準拠
- **ベーステンプレート**: `templates/page-template.html`
- **ヒーローセクション**: `templates/subpage-hero-template.html` (company カテゴリ)
- **カードシステム**: `css/card-system.css` の `.image-card` をベースに `.member-card` を作成

### グリッドレイアウト
```css
.members-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: var(--spacing-8);
}

@media (max-width: 768px) {
  .members-grid {
    grid-template-columns: repeat(2, 1fr);
    gap: var(--spacing-4);
  }
}

@media (max-width: 480px) {
  .members-grid {
    grid-template-columns: 1fr;
  }
}
```

## 完了条件

### デザイン
- [ ] デザイントークンのみ使用
- [ ] 768px/1024pxで崩れない
- [ ] メンバーカードが4列→2列→1列にレスポンシブ対応

- [ ] Lighthouse (Accessibility) 90+ 目安
- [ ] alt属性が適切に設定
- [ ] 見出し階層が正しい
- [ ] キーボードナビゲーション対応

- [ ] 画像はwebp優先 (既存jpg使用も可)
- [ ] 読み込み速度最適化
